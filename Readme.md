# DUXAPP
duxapp是基于Taro开发的模块化的前端框架，同时支持小程序、H5、ReactNative。

## 创建

```bash
npx duxapp-cli create 项目名称
```

项目初始化后将会自动安装依赖，安装完成就可以打开项目进行开发了

## 当前项目使用

```bash
# install
yarn
# build debug apk
yarn debug:android --app skiaCanvas
# start metro
yarn start --app skiaCanvas

# H5
yarn dev:h5 --app skiaCanvas
# 小程序
yarn dev:weapp --app skiaCanvas
```
